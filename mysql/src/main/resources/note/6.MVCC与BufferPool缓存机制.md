# MVCC 多版本并发控制机制
前面提到过，MySQL 是通过 MVCC(Multi-Version Concurrency Control) 来实现事务的隔离性的。通过这种机制避免了频繁加锁互斥，提高了 MySQL 读写的效率。
MySQL 事务在读已提交和可重复读隔离级别下实现了 MVCC 机制。
